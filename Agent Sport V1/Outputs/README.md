# Outputs — Récapitulatif des sessions

> Ce dossier contient les outputs générés lors des sessions de coaching.
> Un fichier par session si l'output mérite d'être sauvegardé (programme personnalisé, plan semaine, diagnostic).

---

## Format des fichiers d'output

```
Outputs/
└── [YYYY-MM-DD_type]/
    ├── programme.md      ← programme de séance généré
    ├── plan_semaine.md   ← plan hebdomadaire complet
    └── diagnostic.md     ← analyse stagnation, blessure, etc.
```

## Quand sauvegarder un output ?

- Programme de séance personnalisé (adapté à l'état du jour + charges actuelles)
- Plan sur 4-8 semaines
- Diagnostic de stagnation avec recommandations
- Plan de préparation pour une compétition ou un objectif précis
- Bilan mensuel de progression

---

## Note

Les outputs simples (réponses courtes) ne nécessitent pas d'être sauvegardés ici.
Seuls les documents structurés qui ont de la valeur sur la durée méritent un fichier.
