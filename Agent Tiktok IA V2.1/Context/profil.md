## CE QUI MARCHE — ENSEIGNEMENTS TERRAIN

Basé sur les 4 premiers scripts avec stats complètes :

### Le hook est la variable numéro 1
| Hook | Rétention 5s |
|------|-------------|
| "Le président le plus puissant du monde est mort en **8 secondes**." | **55%** ✅ |
| "On l'a empoisonné. Noyé. Fusillé." | 47% |
| "Le président qui a aboli l'esclavage est tué **5 jours** après sa victoire." | 44% |
| "Une famille entière a disparu en une nuit." | 36% ❌ |

**Règle validée terrain** : hook avec chiffre précis + fait impossible = meilleure rétention à 5s.

### Formule de hook validée
```
[Personnage connu] + [action impossible] + [chiffre ultra-précis]
```

### Structure optimale (basée sur JFK — best performer)
```
Hook (0-3s)       → stat précise + fait impossible
Ancrage (3s)      → date + lieu (max 2 lignes)
Chronologie       → phrases courtes, chiffres précis, rythme haché
Twist (45-55s)    → "Ce que personne ne sait..." ou "Et si X avait..."
Leçon (55-60s)    → phrase philosophique courte
Signature         → "L'histoire a ses leçons. À vous d'en tirer les vôtres."
CTA débat         → question qui invite au commentaire / à la théorie
```

### Durée optimale
- ✅ **60s = optimal** (JFK best performer)
- ✅ 55-63s = acceptable
- ❌ 71s = trop long, complétion chute

### Angles qui percent
1. Assassinats de personnages historiques majeurs (JFK, Lincoln)
2. Personnages quasi-immortels / impossibles à tuer (Raspoutine)
3. Secrets cachés dans des faits connus (diamants dans les vêtements)
4. Angle complot / vérité officielle remise en question
5. Twist hypothétique ("Et si X avait survécu ?")
