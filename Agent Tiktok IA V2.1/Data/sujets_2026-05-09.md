## PHASE 3 — NOTATION (/10)

### Sujet 1 — Jesse James / Robert Ford ⭐ RECOMMANDÉ (Score 9.5/10)

**Sujet** : Jesse James — Hors-la-loi américain (1847-1882) / Robert Ford — traître (1861-1892)
**Angle** : 3 avril 1882. Robert Ford abat Jesse James dans le dos avec le pistolet que Jesse lui avait offert. Ford cherchait la gloire → il est devenu pour toujours "le lâche qui a tué dans le dos." 10 ans plus tard, Ford est lui-même tué dans le dos dans son propre bar.

| Critère | Note | Justification |
|---------|------|--------------|
| Potentiel viral | 2.5/2.5 | Far West = universellement connu 15-24 FR, film, culture pop ✅ |
| Twist ironique cruel identifiable | 2.5/2.5 | **PARFAIT** : Ford cherchait la gloire → honte éternelle → tué pareil 10 ans après ✅ |
| WTF moment avant 8s | 2/2 | Pistolet offert par la victime + tué dans le dos = double WTF immédiat ✅ |
| Potentiel de débat / partages | 1.5/1.5 | "Ford méritait-il ce qui lui est arrivé ?" = débat éthique ouvert ✅ |
| Angle unique disponible | 1/1.5 | Connu dans la culture pop mais l'angle "mort miroir" rarement traité en FR ✅ |

**Twist ironique cruel identifié** :
> Ford voulait la gloire en tuant Jesse James → il est devenu éternellement "le lâche qui a tiré dans le dos" → forcé de rejouer le meurtre sur scène chaque soir → lui-même tué dans le dos 10 ans plus tard dans son propre bar.

*Formule* : [Ford cherchait la gloire] → [sa méthode (tirer dans le dos) est devenue son identité maudite, puis son destin]

**Pourquoi ça va marcher** :
- Far West = esthétique immédiate pour 15-24 ans (films, jeux, culture US)
- Double WTF : pistolet offert par la victime + mort miroir 10 ans après
- Twist ironique cruel le plus fort de toute la base de données : équivalent Mithridate
- Angle "loyauté / trahison" = universel, génère des commentaires longs
- "Ford méritait-il ce qui lui est arrivé ?" = débat éthique ouvert garanti

**Risques / points de vigilance** :
- Sujet américain (Far West) → s'assurer que les détails sont précis (dates, noms)
- Personnage moins connu en France que JFK → compenser avec un hook ultra-fort sur la trahison
- Robert Ford = personnage principal, pas Jesse James → angle inhabituel mais plus original

**Idée de hook** :
> "Il a tué Jesse James dans le dos.
> 10 ans plus tard... quelqu'un l'a tué pareil."

**Ce qui va faire réagir les gens** :
- "Ford méritait-il ce qui lui est arrivé ?" → débat éthique, commentaires longs
- "Tué avec l'arme offerte par sa victime" = WTF partageable
- La symétrie parfaite de la mort = image mentale puissante

### Sujet 2 — Mata Hari (Score 8.8/10)

**Sujet** : Mata Hari — danseuse, soi-disant espionne (1876-1917)
**Angle** : Fusillée pour espionnage au profit de l'Allemagne le 15 octobre 1917. Problème : les Allemands eux-mêmes avaient envoyé des messages codés pour la piéger. Les preuves étaient inexistantes. Le procureur lui-même a dit "il n'y avait pas de quoi fouetter un chat."
**Twist ironique cruel** : Espionnait pour la France ET l'Allemagne → les deux l'ont trahie. L'Allemagne l'a piégée. La France l'a exécutée.
**Hook** : "Elle espionnait pour 2 pays à la fois. Les 2 l'ont trahie."
**Risques** : Féminisme + espionnage = sensible. Sujet déjà traité dans la culture pop.
