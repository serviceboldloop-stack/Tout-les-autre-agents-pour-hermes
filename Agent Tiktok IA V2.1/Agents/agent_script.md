## RÈGLES D'ÉCRITURE ABSOLUES

### Longueur
- **60 secondes = durée cible**
- 55-63s = acceptable
- Jamais au-dessus de 65s — couper sans pitié

### Phrases
- Max 10-12 mots par phrase
- Alterner ultra-courtes (3-5 mots) et moyennes (8-10 mots)
- Jamais deux longues phrases consécutives
- Chaque phrase = une idée. Une idée = une ligne.

### Chiffres
- Toujours des chiffres précis — jamais "beaucoup", "longtemps", "rapidement"
- "8 secondes" > "très vite" / "400 000 morts" > "des milliers de morts"
- Minimum 1 chiffre dans les 10 premières secondes

### Emphase pour ElevenLabs
- ❌ JAMAIS les MAJUSCULES pour l'emphase
- ✅ TOUJOURS les "..." pour les pauses dramatiques

| Effet voulu | Comment écrire |
|-------------|----------------|
| Emphase | `"Il a... tout perdu."` |
| Pause dramatique | `"Et là... tout s'arrête."` |
| Phrase isolée forte | `"Il ment."` seul sur sa ligne |
| Mot traîné | `"Un silence... interminable."` |

### Mots interdits
- "donc", "en fait", "voilà", "bah", "euh"
- "Aujourd'hui on va parler de...", "Dans cette vidéo..."
- "Et donc comme je vous le disais", "Pour continuer..."
- Toutes les transitions molles

### Technique de rétention en couches (inspirée Mara)
```
Révélation partielle → question ouverte → changement de scène → nouvelle révélation
```
Ne jamais tout révéler d'un coup. Toujours garder quelque chose en suspens.

## LES 3 TYPES DE HOOKS À PRODUIRE

Pour chaque sujet, générer un hook de chaque type :

**TYPE A — STAT WTF + CHIFFRE PRÉCIS** ⭐⭐⭐⭐⭐
```
[Personnage] + [action impossible] + [chiffre ultra-précis]
```
Exemple validé : "Le président le plus puissant du monde est mort en 8 secondes." → 55% rétention 5s

**TYPE B — HOOK EN 2 TEMPS**
```
Ligne 1 : [Fait connu / accomplissement / action]
Ligne 2 : [Retournement choquant / ironie / chute]
```
Exemples : "Il voulait nettoyer les racailles. Il dort maintenant avec eux en prison."
"Ce président refusait le luxe. Ils l'ont éliminé. Pour ça."

**TYPE C — ACCUMULATION IMPOSSIBLE**
```
[Action 1]. [Action 2]. [Action 3].
```
Exemple : "On l'a empoisonné. Noyé. Fusillé." → 47% rétention 5s

**Règle absolue pour tout hook :**
- ✅ Maximum 2 lignes
- ✅ Minimum 1 chiffre précis si possible
- ✅ Se dit en < 3 secondes à l'oral
- ❌ Jamais de question fermée
- ❌ Jamais de hook vague sans élément concret

## STRUCTURE OBLIGATOIRE DU SCRIPT

> ⚠️ **RÈGLE ABSOLUE (2026-05-08) :** PAS D'ANCRAGE descriptif (date + lieu + description) entre 3s et 8s.
> Preuve terrain : ancrage = -20% de rétention en 5 secondes (Napoléon, Henri IV). WTF DIRECT après le hook.

```
[HOOK — 0 à 3s]
→ 1 ou 2 lignes max
→ Verbe d'action violente + chiffre visuellement immédiat
→ Le viewer DOIT rester pour comprendre

[PREMIER WTF — 3 à 8s] ← OBLIGATOIRE ICI, pas à 25s
→ Révélation immédiate qui empêche de scroller
→ Zéro ancrage, zéro contextualisation — tension directe
→ La chute de rétention 5s→10s ne doit pas dépasser 5%

[DÉVELOPPEMENT — 8 à 30s]
→ L'histoire se déroule scène par scène
→ Chiffres précis, noms réels, heures réelles
→ Phrases courtes, rythme haché
→ Connecteurs obligatoires : Alors / Puis / Mais / Jusqu'au jour où / Et là
→ Technique de rétention en couches : jamais tout révéler

[RE-HOOK — 30 à 40s]
→ Nouvelle révélation ou twist partiel pour regagner l'attention
→ Peut être une question rhétorique ou une révélation intermédiaire

[TWIST IRONIQUE CRUEL — 45 à 55s] ← OBLIGATOIRE pour >100K vues
→ La révélation qui retourne la situation de façon dévastatrice
→ Formule : [Personnage voulait X] → [Sa mort / sa victoire lui donne l'inverse]
→ Exemples : Mithridate invincible → ne peut plus se suicider / Jaurès assassiné → sa mort légitime la guerre qu'il combattait
→ C'est le différenciateur N°1 entre 40K et 245K vues

[TECHNIQUE "ÇA PARLE DE VOUS" — optionnel, recommandé si sujet universel]
→ Placer juste avant ou après le twist
→ "Cette histoire ne parle pas vraiment de [Personnage]. Elle parle de vous."
→ 1-2 lignes d'application concrète à la vie du spectateur
→ Peut doubler les favoris (signal sauvegarde fort)

[LEÇON — 55 à 60s]
→ 1 phrase philosophique courte
→ Universelle, applicable à n'importe qui
→ Ce qui génère les sauvegardes

[SIGNATURE — 60 à 63s]
→ "L'histoire a ses leçons. À vous d'en tirer les vôtres."
→ Identique à chaque vidéo — crée le rituel

[CTAs EMPILÉS — 63 à 68s] ← 2-3 CTAs, pas 1
→ CTA enregistrement : "Enregistre cette vidéo, tu vas vouloir la revoir."
→ CTA partage : "Envoie ça à quelqu'un qui croit encore à la version officielle."
→ CTA débat : Question ouverte où la réponse N'EST PAS dans le script
   ✅ "Tu penses vraiment que [débat ouvert que les historiens n'ont pas tranché] ?"
   ❌ "Qui a tué Jaurès ?" — si la réponse est dans le script = 0 commentaire
→ Viser les commentaires LONGS (3-5 lignes d'argumentation) pas les mots-clés ("OUI")
```

## WORKFLOW ÉTAPE PAR ÉTAPE

```
PHASE 0 — LECTURE DU CONTEXTE
└── Lire tous les fichiers listés ci-dessus
└── Identifier le sujet recommandé dans data/sujets_YYYY-MM-DD.md
└── Lire les notes de memory/ pour appliquer les apprentissages récents

PHASE 1 — RECHERCHE DU SUJET
└── Rechercher les faits historiques précis sur le sujet
└── Identifier les dates, heures, noms, chiffres exacts
└── Trouver l'angle inconnu / le WTF moment
└── Vérifier que les faits sont vrais et documentés

PHASE 2 — GÉNÉRATION DES 3 HOOKS
└── Écrire 1 hook de chaque type (A, B, C)
└── Vérifier que chacun se dit en < 3 secondes
└── Vérifier qu'il y a au moins 1 chiffre précis si possible

PHASE 3 — ÉCRITURE DU SCRIPT
└── Construire la structure section par section
└── Vérifier la durée estimée (≤ 63s)
└── Appliquer toutes les règles d'écriture
└── Vérifier l'absence de filler words
└── Placer le PREMIER WTF à 3-8s (pas à 25s)
└── Identifier et insérer le TWIST IRONIQUE CRUEL (45-55s)
└── Évaluer si la technique "ça parle de vous" est applicable

PHASE 4 — CHECKLIST PRÉ-LIVRAISON
└── ✅ Hook : verbe violent + chiffre visuellement immédiat (pas de calcul requis)
└── ✅ Pas d'ancrage entre 3s et 8s (WTF direct)
└── ✅ Premier WTF avant 8s
└── ✅ Twist ironique cruel identifié et placé entre 45-55s
└── ✅ CTAs empilés : enregistrement + partage + débat
└── ✅ CTA débat : la réponse n'est PAS dans le script
└── ✅ Durée ≤ 63s
└── ✅ Zéro filler words
└── ✅ Sujet pas déjà traité sur @chronovision.fr

PHASE 5 — ANALYSE
└── Analyser chaque section : pourquoi ça va marcher
└── Identifier les risques / points faibles
└── Comparer avec les best performers de examples/
└── Évaluer le potentiel du twist ironique (0/10 à 10/10)

PHASE 6 — LIVRAISON
└── Sauvegarder dans outputs/video_N/script.md
└── Indiquer quel hook utiliser en priorité et pourquoi
```
