## RÉPONSE À LA QUESTION DE LUCAS : 30 SECONDES OU 60 SECONDES ?

**Les données de cette vidéo justifient le test 30 secondes. Voici pourquoi :**

| Métrique | Vidéo 60s actuelle | Si vidéo 30s (estimation) |
|----------|--------------------|--------------------------|
| Complétion | 11.83% | ~33% (les 33% qui tiennent à 30s voient tout) |
| Signal algo | ❌ faible | ✅✅ fort (x3 le signal de complétion) |
| Twist visible | ✅ par 33% | ✅ par 33% (inchangé) |
| CTA vu | ❌ seulement 12% | ✅ par 33% |

**Conclusion : même nombre de personnes voient le twist, mais la complétion triple.** TikTok interprète ça comme une vidéo excellente et l'amplifie davantage.

**MAIS attention : la structure doit changer.** Un format 30s ne peut pas être une version raccourcie d'un script 60s. Il faut écrire pour 30s dès le départ :
```
Hook (0-3s) → WTF direct et COMPLET (3-8s) → Mini-développement express (8-18s) → Twist ironique cruel (18-26s) → 2 CTAs (26-30s)
```
Mussolini en 30s = possible. Le twist (même lieu, 9 mois après) peut tenir en 20 secondes.

## RECOMMANDATIONS POUR LA PROCHAINE VIDÉO

**Priorité 1 — Tester le format 30 secondes**
→ Preuve : complétion 11.83% à 60s vs ~33% estimé à 30s = signal algo x3 potentiel
→ Solution : écrire un script structuré pour 30s dès le départ (pas de coupage d'un script 60s)
→ Impact attendu : complétion > 25%, meilleure distribution initiale par l'algo

**Priorité 2 — WTF à 3-8s : donner le référent AVANT la révélation**
→ Preuve : chute -13% entre 5s et 10s = le viewer ne comprend pas "ce lieu" sans contexte
→ Solution : nommer le lieu ou montrer l'image AVANT de dire "c'est lui-même qui l'avait choisi"
→ ✅ "Piazzale Loreto, Milan. C'est là qu'il est mort. Et c'est lui qui avait choisi ce lieu — pour exposer ses ennemis."
→ ❌ "Et ce lieu… il l'avait lui-même choisi." (trop abstrait au 4e seconde)
→ Impact attendu : rétention 10s remonte de 42% vers 50%+

**Priorité 3 — Développement 8-20s : ACTION, pas contexte**
→ Preuve : chute -7% entre 10s et 20s = "22 ans de pouvoir absolu, se rêve en César" = ancrage déguisé
→ Solution : après le WTF, enchaîner sur une ACTION ou une tension, pas une biographie
→ ✅ "Il fuit. Déguisé en soldat allemand. Dans un camion. Les partisans fouillent les convois."
→ ❌ "Mussolini. 22 ans de pouvoir absolu. L'Italie entière à ses pieds."
→ Impact attendu : rétention 20s remonte vers 40%+

**Priorité 4 — Durée max 60s strict (idéal 30s en test)**
→ Preuve : post-60s chute de -11% = la section CTA à 63-68s n'est vue que par 12%
→ Solution : couper à 60s max, idéalement 30s en format test
→ Impact attendu : complétion remonte mécaniquement

**Priorité 5 — Hashtag : vérifier la liste avant publication**
→ 1 hashtag supprimé pour violation = réduction distribution probable
→ Solution : utiliser uniquement des hashtags de niche validés, éviter les termes trop politiques/historiquement sensibles liés au fascisme (TikTok est très sensible à ce vocabulaire)
→ Recommandé : #histoire #histoirefrancaise #storytelling #documentaire + 1 hashtag sujet neutre
