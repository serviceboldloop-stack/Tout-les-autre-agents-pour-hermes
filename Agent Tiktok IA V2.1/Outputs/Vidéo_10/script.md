## LES 3 HOOKS

**HOOK A — Ironie de conséquence + chiffre ⭐ RECOMMANDÉ**

> Jugé en 55 minutes. Fusillé le jour de Noël."

📊 Pourquoi : Ligne 1 = ironie immédiate (le dictateur se prend pour un père). Ligne 2 = choc double : "55 minutes" (chiffre précis visuellement immédiat) + "jour de Noël" (contexte culturel explosif — tout le monde a une image de Noël, personne ne l'associe à une exécution). La dissonance "Père du peuple → fusillé par son peuple le jour de Noël" est maximale. Test : Chiffre ✅ / Tension ✅ / Mystère ✅ / <3s ✅

**HOOK B — Stat WTF + chiffre précis**
> "24 ans de dictature.
> Jugé et condamné à mort en 55 minutes."

📊 Pourquoi : Disproportion pure (24 ans vs 55 minutes) = fait impossible à croire. "24 ans" + "55 minutes" = deux chiffres précis qui créent une dissonance chiffrée immédiate. Plus analytique que A, moins émotionnel. Test : Chiffre ✅ / Tension ✅ / Mystère ✅ / <3s ✅

**HOOK C — Révélation mystérieuse**
> "Il se croyait intouchable.
> Le 25 décembre 1989, son propre peuple l'a fusillé."

📊 Pourquoi : "Se croyait intouchable" = universel, tout le monde a connu quelqu'un comme ça. "Son propre peuple" + "25 décembre" = double choc. Moins de chiffre précis dans les 3s, moins fort que A sur la rétention 5s.

**⭐ HOOK RECOMMANDÉ : A**
Raison : "55 minutes" est visuellement immédiat (règle #13 validée terrain), "Noël" crée un choc culturel que B n'a pas. La dissonance "Père du peuple → fusillé le jour de Noël" est plus émotionnelle que la disproportion chiffrée pure de B.
