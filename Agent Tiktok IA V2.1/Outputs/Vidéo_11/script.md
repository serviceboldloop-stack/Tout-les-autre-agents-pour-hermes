## SUJET & ANGLE

**Sujet** : Alan Turing — mathématicien britannique (1912–1954)
**Angle** : L'homme qui a cassé Enigma et raccourci la Seconde Guerre mondiale de plusieurs années a été condamné par son propre pays pour homosexualité, castré chimiquement, et est mort empoisonné à 41 ans.
**WTF moment** : L'État qu'il avait sauvé l'a lui-même détruit. Condamné 8 ans après la victoire.
**Twist ironique cruel** : [Voulait sauver l'Europe des nazis] → [Son propre gouvernement le condamne, le torture chimiquement, l'efface — puis attend 61 ans pour s'excuser]
**Source** : Faits historiques vérifiés — Bletchley Park, procès 1952, mort 7 juin 1954, pardon royal décembre 2013.

## LES 3 HOOKS

**HOOK A — Ironie de conséquence + chiffre ⭐ RECOMMANDÉ**
> "Il a sauvé 14 millions de vies.
> Son pays l'a castré chimiquement pour ça."

📊 Pourquoi : Ligne 1 = fait colossal avec chiffre précis. Ligne 2 = retournement absolu et violent — "castré chimiquement" est une image physique immédiate qui choque. La dissonance est maximale : sauver des millions → être mutilé par les sauvés. Rétention 5s cible : 65%+.

**HOOK B — Espoir → Désastre**
> "8 ans après avoir gagné la guerre... il est arrêté.
> Pas pour espionnage. Pour avoir aimé un homme."

📊 Pourquoi : "8 ans après" crée un décalage temporel fort. "Pas pour espionnage" = contre-attente qui force à rester. Mais manque un chiffre dans les 3 premières secondes — moins percutant que A.

**HOOK C — Accumulation impossible**
> "Il a cassé le code secret nazi. Raccourci la guerre de 4 ans. Puis son pays l'a tué."

📊 Pourquoi : Accumulation d'accomplissements → chute brutale. Court, rythme fort. Mais "tué" est moins précis que "castré chimiquement" — moins de choc visuel.

**⭐ HOOK RECOMMANDÉ : A**
Raison : "14 millions de vies" + "castré chimiquement" = les deux éléments les plus choquants réunis en 2 lignes. La violence de la ligne 2 sur le fond héroïque de la ligne 1 crée une dissonance cognitive immédiate. Dit en ~3s. ✅

## ANALYSE PAR SECTION

**Hook**
→ "14 millions de vies" = chiffre précis validé terrain (règle #3). "Castré chimiquement" = violence physique concrète qui choque et arrête le scroll. Turing est connu des 15-24 FR (enseigné au lycée, film The Imitation Game, test de Turing). Risque "qui c'est ?" = faible. Cible rétention 5s : 65%+.

**Structure et rythme**
→ Phrases nominales systématiques dans le développement ("Solitaire. Brillant. Différent.") = rythme haché, tension. "La Bombe." seul sur sa ligne = pause dramatique forte. Séries courtes tout au long. Connecteurs : "Alors", "Et c'est là que", "2 ans plus tard".

**Premier WTF (3-8s)**
→ Timing : le WTF arrive à ~4s. "Pas pour espionnage. Pour avoir aimé un homme." = révélation qui retourne l'estomac immédiatement. Pas d'ancrage entre 3s et 8s ✅. Chute 5s→10s estimée : < 5% ✅.

**Twist ironique cruel**
→ Formulation exacte : [Turing voulait sauver l'Europe → il l'a sauvée] → [L'Europe qu'il a sauvée l'a condamné, castré, effacé]
→ Score d'ironie : **10/10** — symétrie totale. Il donne tout. Il reçoit la destruction. 61 ans d'attente pour des excuses. La pomme au cyanure (qui fait écho à Apple, au symbole technologique) ajoute une couche supplémentaire d'ironie.

**Technique "ça parle de vous"**
→ Non incluse explicitement — l'histoire parle d'elle-même. Optionnel : ajouter entre le twist et la leçon "Cette histoire ne parle pas vraiment de Turing. Elle parle de tous ceux qu'on a détruits pour ce qu'ils étaient." Si la durée le permet.

**CTAs empilés**
→ CTA enregistrement : "Enregistre cette vidéo, tu vas vouloir la revoir." ✅
→ CTA partage : "Envoie ça à quelqu'un qui pense encore que la justice protège tout le monde." ✅ (résonne universellement — injustice que tout le monde reconnaît)
→ CTA débat : "61 ans pour des excuses officielles. C'était assez pour toi ?" ✅ Deux camps garantis : "mieux vaut tard que jamais" vs "une excuse ne ressuscite pas quelqu'un". Réponse non dans le script ✅. Commentaires longs attendus.

**Risques identifiés**
→ "Castré chimiquement" : TikTok peut être sensible. Formuler en voix off de façon clinique (pas de détails visuels) = acceptable.
→ "Aimé un homme" : ne pas utiliser de termes militants qui risquent un shadowban. Le script reste factuel ✅.
→ Durée à surveiller : si ElevenLabs allonge les "...", le segment twist peut déborder. Couper "Et ce n'est pas une exagération." si nécessaire (3s gagnées).

**Durée estimée** : ~62–65 secondes script + ~7s CTAs = ~69–72s total
→ Version courte : supprimer "Et ce n'est pas une exagération." (ligne WTF) — économise ~3s
→ Version serrée cible : 63s script + 5s CTAs = 68s total

**Potentiel viral** : **10/10** — Sujet universellement connu + injustice documentée la plus choquante de la niche (sauver des millions, recevoir la destruction) + twist d'ironie parfait + CTA clivant à fort engagement. Équivalent Mussolini en potentiel. Angle injustice = partages massifs attendus (tout le monde veut montrer ça à quelqu'un).
