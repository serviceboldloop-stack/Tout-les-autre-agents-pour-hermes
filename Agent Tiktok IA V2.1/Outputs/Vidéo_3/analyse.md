## RECOMMANDATIONS POUR LA PROCHAINE VIDEO

**Priorite 1 — Ne jamais refaire un sujet deja traite**
→ Preuve : Lincoln deja fait (470 vues avant), remake a fait 267 vues seulement. Audience blassee = pas de curiosite fraiche.
→ Solution : L'agent research DOIT verifier que le sujet n'a jamais ete traite sur @chronovision.fr. Ajouter un check explicite dans l'agent research.
→ Impact attendu : +500-1000 vues (retour a du sujet frais)

**Priorite 2 — Hook a 60%+ obligatoirement**
→ Preuve : 49% = 267 vues seulement (200-view jail). JFK 55% = 476 vues. Cible 60%+ pour s'echapper du jail.
→ Solution : Chiffre encore plus precis, ou hook Type 2 (espoir → desastre) a tester.
→ Impact attendu : Sortie du jail → 500-1000 vues

**Priorite 3 — Re-hook obligatoire a 3-5s**
→ Preuve : Chute 49% → 37% a 10s = -12%. Le passage ancrage tue la retention.
→ Solution : Ajouter une phrase WTF ou un chiffre de choc a 3-5s, juste apres le hook. Renforcer le connecteur "Alors" → "Mais".
→ Impact attendu : Maintien a 45%+ a 10s

**Priorite 4 — CTA debat sur sujet qui divisera**
→ Preuve : 0 commentaire. Personne n'a d'avis sur un sujet deja connu.
→ Solution : Choisir des sujets ou le complot est NON-CONNU et divise (ex: "Et si Booth avait reussi ?" sur un sujet frais).
→ Impact attendu : 3-5 commentaires minimum

**Priorite 5 — Duree max 60s stricte**
→ Preuve : 62s → 18,22% completion (sous les 20% cible).
→ Solution : Couper 2s dans le developpement. Viser 58-60s max.
→ Impact attendu : +2-3% completion

**Priorite 6 — A CONSERVER : Chiffres precises**
→ Ce qui a bien marche : "5 jours" dans le hook = 49% (mieux que 44% avant). Le format fonctionne.
→ Garder : Toujours un chiffre dans les 10 premieres secondes.
