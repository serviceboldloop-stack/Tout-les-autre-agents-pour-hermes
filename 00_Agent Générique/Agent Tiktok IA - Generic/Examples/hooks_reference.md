## PRINCIPE FONDAMENTAL — LE DOUBLE HOOK

Le système validé terrain pour [TON_COMPTE] :

```
Ligne 1 → Fait choc / chiffre précis / accomplissement → ARRÊTE le scroll
Ligne 2 → Retournement / ironie / révélation abstraite → FORCE à rester pour comprendre
```

Le viewer doit être dans cet état après 3 secondes :
**"Attends... comment c'est possible ? Je dois rester pour comprendre."**

Si la ligne 2 ne crée pas cette tension avec la ligne 1, le hook est raté.

## LES 6 TYPES DE DOUBLE HOOK

---

### TYPE 1 — IRONIE DE CONSÉQUENCE ⭐⭐⭐⭐⭐
**Formule :** `[Personnage] + [action A] / [conséquence B qui la trahit ou l'annule]`

La ligne 2 révèle que l'action de la ligne 1 a produit le résultat inverse ou une conséquence catastrophique inattendue.

**Exemples validés :**
```
"Cet homme a convaincu 79 millions de personnes de partir en guerre.
60 millions ne sont jamais revenus."
— @histoires.vrais.a (série WW2)
```
```
"Le président qui a aboli l'esclavage. Tué 5 jours après sa victoire."
— [TON_COMPTE] (Lincoln, 44% rétention 5s)
```
```
"Il voulait sauver son pays de la pauvreté.
Il vivait dans 1 palais doré de marbre."
— @histoires.vrais.a (Mobutu)
```

**Pourquoi ça marche :** La ligne 1 crée une attente. La ligne 2 la détruit. Le viewer est dans un état de dissonance cognitive — il doit regarder pour résoudre.

**À utiliser quand :** Le personnage a fait quelque chose de grand qui a mal tourné, ou dont les conséquences contredisent l'intention.

---

### TYPE 2 — ESPOIR → DÉSASTRE ⭐⭐⭐⭐⭐
**Formule :** `[Croyance/espoir des victimes, image familière] / [Réalité chiffrée et brutale]`

La ligne 1 est douce, humaine, universelle. La ligne 2 est un coup de massue.

**Exemples validés :**
```
"Ces soldats pensaient qu'ils seraient rentrés pour Noël.
20 millions d'entre eux ont fini en enfer."
— @histoires.vrais.a (WW1)
```
```
"Il pensait qu'une seule balle suffirait.
Il en a reçu 8."
— [À créer pour un sujet d'assassinat]
```

**Pourquoi ça marche :** "Noël" est une image universelle. Tout le monde a fait une promesse de retour. La trahison de cette promesse est ressentie viscéralement avant même de savoir de quoi il s'agit.

**À utiliser quand :** Les victimes de l'histoire avaient une croyance naïve avant que tout bascule.

---

### TYPE 3 — STAT WTF + CHIFFRE PRÉCIS ⭐⭐⭐⭐⭐
**Formule :** `[Personnage connu] + [action impossible] + [chiffre ultra-précis]`

Un seul chiffre très précis qui rend le fait incroyable.

**Exemples validés :**
```
"Le président le plus puissant du monde est mort en 8 secondes."
— [TON_COMPTE] (JFK, 55% rétention 5s ⭐ BEST PERFORMER)
```
```
"L'homme qui a conquis l'Europe est mort avec 38 fois trop d'arsenic dans les cheveux."
— [TON_COMPTE] (Napoléon)
```
```
"Il a posé une bombe à 2 mètres d'Hitler."
— [TON_COMPTE] (Stauffenberg, Hook A)
```

**Pourquoi ça marche :** Le chiffre précis crédibilise l'impossible. "8 secondes" > "très rapidement". "38 fois" > "beaucoup trop d'arsenic". La précision = authenticité = confiance = rétention.

**Règle absolue :** Au moins 1 chiffre précis dans TOUT hook [TON_COMPTE]. Même si le hook est d'un autre type, trouver comment y insérer un chiffre.

---

### TYPE 4 — MISÈRE → ASCENSION ⭐⭐⭐⭐
**Formule :** `[Image de départ humble/sordide très concrète] / [Identité finale spectaculaire]`

Le contraste entre les deux lignes est si extrême que le viewer ne voit pas comment le trajet est possible.

**Exemples validés :**
```
"Cette femme dormait dans de la paille au milieu des rats.
Elle a conquis toute la France."
— @histoires.vrais.a (Comtesse du Barry)
```
```
"Elle se faisait payer quelques pièces en échange de son corps.
C'est devenu la reine des pirates."
— @histoires.vrais.a (Ching Shih)
```
```
"Cette danseuse de rue est devenue l'impératrice la plus puissante du monde."
— @histoires.vrais.a (Théodora)
```
```
"À 20 ans, il était déjà médecin, pilote et millionnaire. Sans aucun diplôme."
— @nonametales (Frank Abagnale)
```

**Pourquoi ça marche :** Le trajet de A vers B est incompréhensible. La seule façon de le comprendre = regarder jusqu'à la fin.

---

### TYPE 5 — RÉVÉLATION MYSTÉRIEUSE ⭐⭐⭐⭐
**Formule :** `[Fait connu] / ["Ce que personne ne sait..." ou contexte qui rend le fait mystérieux]`

**Exemples validés :**
```
"Il était l'influenceur le plus riche d'Instagram.
Jusqu'à ce que le FBI s'en mêle."
— @nonametales (Hush Puppi)
```
```
"Ce président se déplaçait en vélo et refusait le luxe.
Ils l'ont éliminé. Pour ça."
— @histoires.vrais.a (Thomas Sankara)
```

**Pourquoi ça marche :** "Pour ça." en fin de ligne 2 crée une tension maximale avec un minimum de mots. Le viewer doit rester pour comprendre le "ça."

**Variantes de clôture de ligne 2 :**
- "Pour ça." → tension court
- "Jusqu'à ce que..." → rupture annoncée
- "Ce que personne ne sait..." → secret promis
- "Et personne ne l'a jamais cru." → complot ouvert

---

### TYPE 6 — ACTION PARADOXALE ⭐⭐⭐⭐
**Formule :** `[Action physique improbable ou symbolique] / [Impact inattendu]`

L'action de la ligne 1 est visuellement bizarre ou symboliquement forte. La ligne 2 révèle que cette action a eu des conséquences réelles.

**Exemples validés :**
```
"Cette femme a porté 1 barbe pour devenir pharaon.
Son nom est resté dans l'histoire."
— @histoires.vrais.a (Hatchepsout)
```
```
"Cette femme a pris l'apparence de son frère jumeau pour régner.
L'histoire a effacé son nom."
— @histoires.vrais.a
```
```
"On l'a empoisonné. Noyé. Fusillé."
— [TON_COMPTE] (Raspoutine, 47% rétention 5s)
```

**Pourquoi ça marche :** L'action de la ligne 1 est visuelle et bizarre → elle crée une image mentale forte. La ligne 2 donne le contexte ou l'impact.

## RÈGLES ABSOLUES POUR TOUT HOOK

### ✅ Ce qu'il FAUT
- Maximum 2 lignes (3 acceptables si la 3e est un titre fort)
- Au moins 1 chiffre précis si possible (obligatoire pour [TON_COMPTE])
- La ligne 2 crée une tension avec la ligne 1
- Se dit en < 3 secondes à l'oral
- Personnage identifiable ou sujet qui intrigue
- Le viewer doit AVOIR BESOIN de regarder la fin pour résoudre la tension

### ❌ Ce qu'il NE FAUT PAS
- "Bonjour", "Salut", "Aujourd'hui on va voir"
- Question fermée (réponse oui/non)
- Hook vague sans élément concret
- "L'histoire incroyable de..." → dire l'incroyable directement
- Deux lignes qui disent la même chose (pas de contraste)
- Ligne 2 qui explique la ligne 1 (ça enlève la tension)

## EXEMPLES DE HOOKS RATÉS ET LEURS CORRECTIONS

### Cas 1 — Hook sans chiffre ni tension
❌ `"Une famille entière a disparu en une nuit. On n'a retrouvé les corps que 70 ans plus tard."` → Ligne 1 vague + ligne 2 qui explique mais n'intensifie pas
✅ `"Les balles ont rebondi sur les enfants du Tsar. Voilà pourquoi."` → Fait impossible + mystère

### Cas 2 — Ligne 2 qui ne crée pas de tension
❌ `"Il a posé une bombe à 2 mètres d'Hitler. Une table en chêne a changé l'histoire."` → "Une table a changé l'histoire" est trop abstrait comme révélation
✅ `"Il a posé une bombe à 2 mètres d'Hitler. Une table de bois l'a sauvé."` → La chute est concrète et absurde

### Cas 3 — Ligne 1 trop générale
❌ `"Cet homme a changé le monde entier."` → Trop vague, pas de tension
✅ `"Cet homme a convaincu 79 millions de personnes de partir en guerre."` → Chiffre + action précise = WTF immédiat

## FORMULE DE VALIDATION AVANT DE VALIDER UN HOOK

Après avoir écrit un hook, répondre à ces 4 questions :
1. **Chiffre ?** → Y a-t-il au moins 1 chiffre précis ?
2. **Tension ?** → La ligne 2 crée-t-elle une ironie ou un contraste avec la ligne 1 ?
3. **Mystère ?** → Le viewer a-t-il BESOIN de regarder la suite pour résoudre la tension ?
4. **3 secondes ?** → Le hook se dit-il en moins de 3 secondes à 2,5 mots/s ?

Si la réponse à l'une de ces questions est NON → réécrire.
