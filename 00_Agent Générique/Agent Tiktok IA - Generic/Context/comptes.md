## RÈGLES DE SCRIPT VALIDÉES TERRAIN

### Formule de hook qui marche
```
[Verbe d'action violent] + [Personnage / Sujet] + [Chiffre visuellement immédiat]
```
Exemples validés (niche historique) :
- "Le président le plus puissant du monde est mort en **8 secondes**." → 55% rétention 5s
- "On l'a empoisonné. Noyé. Fusillé." → 47% rétention 5s
- Hook vague sans chiffre → < 40% rétention 5s ❌

### Structure optimale (mise à jour 2026-05-09)
```
Hook (0-3s)              → Verbe violent + chiffre immédiat
WTF DIRECT (3-8s)        → Révélation immédiate — PAS d'ancrage
Développement (8-30s)    → Connecteurs narratifs : Alors / Puis / Mais / Jusqu'au jour où
Re-hook (30-40s)         → Nouvelle révélation pour regagner l'attention
Twist ironique (45-55s)  → [Personnage voulait X] → [X le détruit / sa mort sert ce qu'il combattait]
Leçon (55-60s)           → 1 phrase philosophique courte
Signature (60-63s)       → Phrase identique à chaque vidéo
CTAs empilés (63-68s)    → Enregistrement + Partage + Débat (réponse hors du script)
```

### Durée
- ✅ **60s = optimal**
- ✅ 55-63s = acceptable
- ❌ > 65s = trop long, complétion chute
